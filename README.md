# ProDent — Dental Clinic Website Mockup

A static front-end mockup for a dental clinic landing page. Built with plain HTML/CSS/JavaScript — no frameworks.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Home — hero, features, about, services, pricing, calculator, reviews, FAQ, blog preview |
| `about.html` | About — mission, history, team, certificates |
| `services.html` | Services — treatment, surgery, orthodontics, pediatric |
| `blog.html` | Blog — article grid with sidebar |
| `post.html` | Single blog post |
| `contacts.html` | Contacts — form, map, info |
| `404.html` | 404 error page |


## Assets

```
assets/
├── css/style.css   — all styles (~2000 lines)
├── js/main.js      — all JS (menu, counters, calculator, FAQ, AOS, Swiper)
└── img/            — images (placeholders marked with size comments)
```
